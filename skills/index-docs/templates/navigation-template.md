## Usage for AI Agents

When a user asks about {ecosystem}:

1. **Check this file first** - Identify which module likely has the answer
2. **Check capabilities.yaml** - For structured module lookup
3. **Check function_index.yaml** - Find specific function details
4. **WebFetch the URL** - Get live, current documentation for parameters/examples

### Decision Tree

```
User question about {ecosystem}
    │
    ├─ "What function does X?"
    │   └─ Check function_index.yaml by related_tasks
    │
    ├─ "How do I do X?"
    │   └─ Check capabilities.yaml → find module that 'covers' X
    │       └─ WebFetch tutorial or function URL
    │
    ├─ "What are the parameters for X?"
    │   └─ Check function_index.yaml for signature
    │       └─ If more detail needed → WebFetch function URL
    │
    └─ "Compare X and Y approaches"
        └─ WebFetch both function URLs for full details
```
